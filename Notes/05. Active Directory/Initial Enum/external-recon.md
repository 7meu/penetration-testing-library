# External Recon and Enumeration Principles

## Core Principles

* **Passive vs. Active Reconnaissance**
  * **Passive:** Gathering information without directly interacting with the target (public databases, social media).
  * **Active:** Direct interaction with target systems (port scanning, service probing).
* **Scoping:** Defining engagement boundaries.
* **Information Gathering:** Collecting relevant data about the target.
* **Iterative Process:** Reconnaissance continues throughout the assessment.
* **OSINT:** Using publicly available information.

## Key Data Points

* **IP Space:** Network ranges and ASNs.
* **Domain Information:** Domains, subdomains, DNS records.
* **Schema Formats:** Naming conventions for accounts and emails.
* **Data Disclosures:** Public documents or files containing sensitive data.
* **Breach Data:** Leaked or compromised credentials.
