# Essentials

| Tool                                                                                                                                          | Description                                                                                                                                                                                                                                                                       |
| --------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Active Directory Explorer](https://docs.microsoft.com/en-us/sysinternals/downloads/adexplorer)                                               | A tool for browsing and editing Active Directory databases. It allows you to inspect object attributes, take offline snapshots, and compare two snapshots to identify differences in settings, permissions, and objects.                                                          |
| [ADRecon](https://github.com/adrecon/ADRecon)                                                                                                 | A utility that collects a wide range of information from an Active Directory environment and outputs it—often as Excel files—to support high-level reviews and security analysis.                                                                                                 |
| [adidnsdump](https://github.com/dirkjanm/adidnsdump)                                                                                          | A program that enumerates and extracts DNS records from a domain, effectively providing functionality similar to a DNS zone transfer.                                                                                                                                             |
| [BloodHound](https://github.com/BloodHoundAD/BloodHound)                                                                                      | A graph-based tool used to reveal hidden or complex relationships inside Active Directory by importing data collected through SharpHound into a Neo4j-backed graphical interface.                                                                                                 |
| [BloodHound.py](https://github.com/fox-it/BloodHound.py)                                                                                      | A Python implementation of the BloodHound ingestor built on the [Impacket toolkit](https://github.com/CoreSecurity/impacket/), supporting most collection methods and usable from hosts not joined to the domain.                                                                 |
| [C# Inveigh (InveighZero)](https://github.com/Kevin-Robertson/Inveigh/tree/master/Inveigh)                                                    | The C# implementation of Inveigh that provides a partially interactive interface for reviewing intercepted usernames, password hashes, and related data.                                                                                                                          |
| [CVE-2021-1675.py](https://github.com/cube0x0/CVE-2021-1675/blob/main/CVE-2021-1675.py)                                                       | A Python proof-of-concept demonstrating the PrintNightmare vulnerability.                                                                                                                                                                                                         |
| [CrackMapExec (CME)](https://github.com/byt3bl33d3r/CrackMapExec)                                                                             | A post-exploitation and enumeration suite that leverages built-in Active Directory features—such as SMB, WinRM, WMI, and MSSQL—to map environments and automate a variety of attacks.                                                                                             |
| [DomainPasswordSpray.ps1](https://github.com/dafthack/DomainPasswordSpray)                                                                    | A PowerShell script designed to perform password-spraying attacks against domain accounts.                                                                                                                                                                                        |
| [enum4linux](https://github.com/CiscoCXSecurity/enum4linux)                                                                                   | A Linux-based enumeration script that extracts information from Windows and Samba systems.                                                                                                                                                                                        |
| [enum4linux-ng](https://github.com/cddmp/enum4linux-ng)                                                                                       | A modernized and restructured version of Enum4linux that performs similar enumeration tasks using a refined workflow.                                                                                                                                                             |
| [evil-winrm](https://github.com/Hackplayers/evil-winrm)                                                                                       | A tool that provides an interactive PowerShell console over the WinRM protocol on Windows systems.                                                                                                                                                                                |
| [GetNPUsers.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/GetNPUsers.py)                                                | Uses Impacket to enumerate users with the “Do not require Kerberos preauthentication” setting and request their TGTs.                                                                                                                                                             |
| [GetUserSPNs.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/GetUserSPNs.py)                                              | An Impacket tool that finds Service Principal Names linked to regular user accounts, aiding in Kerberoasting attempts.                                                                                                                                                            |
| [gettgtpkinit.py](https://github.com/dirkjanm/PKINITtools/blob/master/gettgtpkinit.py)                                                        | A tool for manipulating certificates and creating Kerberos TGTs using PKINIT.                                                                                                                                                                                                     |
| [getnthash.py](https://github.com/dirkjanm/PKINITtools/blob/master/getnthash.py)                                                              | Uses an existing TGT to request a PAC for the current user through U2U and obtain the NT hash.                                                                                                                                                                                    |
| [gpp-decrypt](https://github.com/t0thkr1s/gpp-decrypt)                                                                                        | Recovers usernames and passwords stored within Group Policy Preferences.                                                                                                                                                                                                          |
| [Group3r](https://github.com/Group3r/Group3r)                                                                                                 | A utility for analyzing Group Policy Objects to discover configuration weaknesses and potential security issues.                                                                                                                                                                  |
| [Hashcat](https://hashcat.net/hashcat/)                                                                                                       | A high-performance password-cracking framework capable of recovering many types of hashes.                                                                                                                                                                                        |
| [Impacket toolkit](https://github.com/SecureAuthCorp/impacket)                                                                                | A Python library containing numerous tools for interacting with various network protocols and performing Active Directory enumeration and attacks.                                                                                                                                |
| [Inveigh.ps1](https://github.com/Kevin-Robertson/Inveigh/blob/master/Inveigh.ps1)                                                             | A PowerShell-based tool that performs spoofing and poisoning of protocols such as LLMNR, NBNS, and mDNS.                                                                                                                                                                          |
| [Kerbrute](https://github.com/ropnop/kerbrute)                                                                                                | A Go application that uses Kerberos pre-authentication to enumerate domain accounts and conduct password spraying or brute-forcing.                                                                                                                                               |
| [LAPSToolkit](https://github.com/leoloobeek/LAPSToolkit)                                                                                      | PowerShell functions that utilize PowerView to audit and exploit environments using Microsoft LAPS.                                                                                                                                                                               |
| [ldapsearch](https://linux.die.net/man/1/ldapsearch)                                                                                          | A command-line tool built into many Linux distributions that interacts with LDAP directories.                                                                                                                                                                                     |
| [lookupsid.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/lookupsid.py)                                                  | A script that brute-forces Security Identifiers (SIDs) to enumerate domain accounts.                                                                                                                                                                                              |
| [Mimikatz](https://github.com/ParrotSec/mimikatz)                                                                                             | A well-known tool capable of extracting plaintext credentials, Kerberos tickets, and performing pass-the-hash attacks.                                                                                                                                                            |
| [mssqlclient.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/mssqlclient.py)                                              | An Impacket script that supports interaction with Microsoft SQL Server instances.                                                                                                                                                                                                 |
| [noPac.py](https://github.com/Ridter/noPac)                                                                                                   | An exploit chain that abuses CVE-2021-42278 and CVE-2021-42287 to elevate privileges from a standard domain account to Domain Admin.                                                                                                                                              |
| [ntlmrelayx.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/ntlmrelayx.py)                                                | An Impacket utility that performs NTLM relay attacks across supported protocols.                                                                                                                                                                                                  |
| [PetitPotam.py](https://github.com/topotam/PetitPotam)                                                                                        | A proof-of-concept exploiting CVE-2021-36942 to coerce Windows hosts into authenticating to other machines through MS-EFSRPC calls.                                                                                                                                               |
| [PingCastle](https://www.pingcastle.com/documentation/)                                                                                       | A security auditing tool that evaluates an AD environment using a risk-rating system based on a maturity model inspired by CMMI.                                                                                                                                                  |
| [PowerView](https://github.com/PowerShellMafia/PowerSploit/blob/master/Recon/PowerView.ps1)/[SharpView](https://github.com/dmchell/SharpView) | PowerShell and .NET tools used to collect Active Directory information, offering functionality similar to built-in Windows `net*` commands while enabling detailed discovery tasks such as checking access, identifying attack paths, and locating Kerberoast/ASREPRoast targets. |
| [psexec.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/psexec.py)                                                        | An Impacket script that emulates PsExec-style remote command execution by spawning a semi-interactive shell.                                                                                                                                                                      |
| [raiseChild.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/raiseChild.py)                                                | An Impacket tool that escalates privileges from a child domain to its parent domain under certain conditions.                                                                                                                                                                     |
| [Responder](https://github.com/lgandx/Responder)                                                                                              | A specialized tool for capturing credentials by poisoning LLMNR, NBNS, and MDNS requests.                                                                                                                                                                                         |
| [rpcclient](https://www.samba.org/samba/docs/current/man-html/rpcclient.1.html)                                                               | A component of the Samba suite that enables a range of enumeration tasks over the remote RPC service in Windows environments.                                                                                                                                                     |
| [rpcdump.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/rpcdump.py)                                                      | An Impacket-based RPC endpoint mapper that lists RPC interfaces exposed on a system.                                                                                                                                                                                              |
| [Rubeus](https://github.com/GhostPack/Rubeus)                                                                                                 | A C# program designed for executing various Kerberos-related attacks and operations.                                                                                                                                                                                              |
| [SharpHound](https://github.com/BloodHoundAD/BloodHound/tree/master/Collectors)                                                               | The C# collector used to gather information about AD objects—such as users, groups, computers, ACLs, GPOs, and sessions—for later ingestion into BloodHound.                                                                                                                      |
| [smbmap](https://github.com/ShawnDEvans/smbmap)                                                                                               | A tool that enumerates SMB shares across a domain and reports accessible resources.                                                                                                                                                                                               |
| [smbserver.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/smbserver.py)                                                  | A simple SMB server implementation that enables easy file transfer and host interaction on Windows networks.                                                                                                                                                                      |
| [Snaffler](https://github.com/SnaffCon/Snaffler)                                                                                              | Scans readable file shares across a domain to identify sensitive data such as credentials and configuration files.                                                                                                                                                                |
| [ticketer.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/ticketer.py)                                                    | A Kerberos ticket creation tool that allows customization of TGTs and TGS tickets.                                                                                                                                                                                                |
| [windapsearch](https://github.com/ropnop/windapsearch)                                                                                        | A Python script automating LDAP queries to list users, groups, and computers within Active Directory.                                                                                                                                                                             |
| [wmiexec.py](https://github.com/SecureAuthCorp/impacket/blob/master/examples/wmiexec.py)                                                      | An Impacket script enabling remote command execution via Windows Management Instrumentation (WMI).                                                                                                                                                                                |
